Key Features

Automated Data Collection

Real-time data collection from multiple tourism sources including INE, TripAdvisor, and Exceltur. Hourly updates ensure the most current information.

PLS-SEM Analysis

Advanced Partial Least Squares Structural Equation Modeling to analyze causal relationships between tourism competitiveness, satisfaction, and employment.

AI-Powered Insights

Intelligent agents using Claude API and local ML models generate contextual recommendations and predictions for tourism policy optimization.

Real-Time Dashboard

Interactive web dashboard with live KPIs, time series visualizations, and the PLS-SEM model diagram. Updates every 5 minutes automatically.

Automated Reporting

Scheduled generation of comprehensive reports with analysis results, AI recommendations, and performance metrics. Export to JSON and CSV formats.

Flexible API

RESTful API endpoints for programmatic access to data, analysis results, and system controls. Enable seamless integration with external systems.

Real-time data flow from collection to intelligent insights and actionable outputs

Data Flow Pipeline

  1. Data Sources: Automated collection from INE (hotel occupancy, employment), TripAdvisor (reviews, rankings), and Exceltur (competitiveness indices)
  2. Processing: Data validation, normalization, and integration into SQLite database
  3. Analysis: PLS-SEM modeling (twice daily) and AI agent analysis (every 2 hours)
  4. Visualization: Real-time dashboard updates with KPIs and interactive charts
  5. Output: Automated reports, API responses, and policy recommendations

Technology Stack

Python 3.8+ Dash/Plotly scikit-learn pandas NumPy SQLite Claude API REST API Bootstrap Git/GitHub

PLS-SEM Model Variables

The system implements a validated structural equation model analyzing relationships between: Tourism Competitiveness (infrastructure, accessibility, sustainability) → Tourism Satisfaction (rankings, reviews, facilities) → Tourism Employment (direct and indirect jobs).

Getting Started

Quick Installation

# Clone the repository
git clone https://github.com/upocuantitativo/aitourism.git
cd aitourism

# Install dependencies (Windows)
install_windows.bat

# Or manually with pip
pip install -r requirements.txt

# Import your data (optional)
python import_excel_data.py data_final.xlsx

# Run the system
python main.py --mode full

# Access the dashboard
# Open browser: http://localhost:8050

System Requirements

  • Python 3.8 or higher
  • 4GB RAM minimum (8GB recommended)
  • 2GB free disk space
  • Internet connection for data sources and AI APIs
  • Modern web browser for dashboard
